## Quick Start
```bash
pip install requests beautifulsoup4
python scripts/analyze.py https://example.com
```
## Scoring
- Named author: 25pts
- Author credentials/bio: 20pts
- Publish date: 20pts
- External references: 20pts
- Schema.org markup: 15pts
